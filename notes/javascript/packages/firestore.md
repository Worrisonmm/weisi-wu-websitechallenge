# The `firestore` Package

The `firestore` package helps us fetch data from a a NoSQL key-value storage option called Firestore from Google. Firestore is available within the Firebase suite of products.


References:

  + [NPM Page](https://www.npmjs.com/package/@google-cloud/firestore)
  + [Docs](https://googleapis.dev/nodejs/firestore/latest/)
  + [Source Code](https://github.com/googleapis/nodejs-firestore)
